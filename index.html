Notable Projects

Image Super-Resolution using Sparsity Priors
Image super-resolution or zooming is simply increasing image dimensions without loss of perceptual quality. This project contains image super-resolution using sparsity priors.
Technologies
image processing
Vehicle Detection and Tracking for Automated Driving Test
This project contains vehicle detection using CDR/CFAR methods and vehicle tracking using Kalman filter, for automation of driving test.
Technologies
estimation, detection
Target Localisation using Time Difference of Arrival
Time difference of arrival (TDOA) is an estimation method for target localisation. The project contains maximum likelihood estimation, best linear estimation and Cramer-Rao bounds.
Technologies
estimation
Spectrum Sensing for Congnitive Radio
Cognitive radio relies on detection of vacant spectral bands for utilisation. This project contains Neymann-Pearson detectors for spectrum sensing.
